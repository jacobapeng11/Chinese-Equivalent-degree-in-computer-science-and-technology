## 二、填空题解答

11. **答案：机器智能**
    - 解析：图灵测试旨在给予机器智能一个满意的操作定义，即如果机器在对话中表现得与人难以区分，则可以认为它具有智能。

12. **答案：LS>1**
    - 解析：在主观贝叶斯方法中，当证据E支持结论H时，似然比LS>1，LS值越大支持程度越高。

13. **答案：模块化、自然性、有效性**
    - 解析：产生式系统的特点包括：模块化（规则独立）、自然性（表达方式接近自然语言）、有效性（便于推理）。

14. **答案：完备、最优**
    - 解析：有界深度优先搜索在深度限制内是完备的（能找到限制深度内的解），但不是最优的（不保证找到最浅的解）。

15. **答案：如果问题有解，算法能保证找到解、找到的解是代价最小的解**
    - 解析：完备性指算法能保证找到解（如果解存在），最优性指找到的解是代价最小的解。

## 三、简答题解答

16. **答案：**
    空子句在归结反演中表示一个永假式，即矛盾。当在归结反演过程中推导出空子句时，说明原假设（即被否定的待证命题）导致了矛盾，根据反证法原理，原假设是错误的，因此原命题（未被否定的命题）是正确的。空子句的出现标志着定理证明的完成。

17. **答案：**
    相同点：都是知识表示方法，都能表示对象和对象之间的关系。
    不同点：
    - 语义网络以节点和弧的图形方式表示知识，直观易懂，但推理机制有限；
    - 框架以结构化方式表示知识，具有槽-侧面系统，支持默认值和继承机制，但结构更复杂；
    - 语义网络更适合表示静态关系，框架更适合表示对象的属性和行为。

18. **答案：**
    信念函数Bel(A)表示对命题A的最小信任程度，即所有支持A的证据之和；似然函数Pl(A)表示对A不怀疑的程度，即不确定但可能支持A的程度。
    关系：Bel(A) ≤ Pl(A)恒成立；Pl(A) = 1 - Bel(¬A)。这表示对A的信任度不超过对A的不怀疑度，且对A的似然度等于对非A的不信任度的补。

19. **答案：**
    LS = P(E|H) / P(E|¬H)，表示证据E在假设H为真时出现的似然性与假设H为假时出现的似然性之比。
    LN = P(¬E|H) / P(¬E|¬H)，表示在假设H为真时证据E不出现的似然性与假设H为假时E不出现的似然性之比。
    LS>1表示E支持H，LS<1表示E反对H，LS=1表示E与H无关；LN含义类似，但针对证据不出现的情况。

## 四、计算题解答

20. **答案：**
    已知：Bel(A)=0.3，Pl(A)=0.7
    
    含义解释：
    - Bel(A)=0.3：对疾病A的信任程度为0.3，即有确凿证据支持A的强度为0.3
    - Pl(A)=0.7：对疾病A不怀疑的程度为0.7，即A可能为真的程度为0.7
    
    根据证据理论性质：Pl(A) = 1 - Bel(¬A)
    所以：Bel(¬A) = 1 - Pl(A) = 1 - 0.7 = 0.3
    
    因此，对非A（即不患病）的信念函数Bel(¬A)为0.3。
    
    这意味着：对"不患疾病A"的信任度为0.3，对"患病A"的信任度为0.3，剩余的0.4表示不确定性（既不信任患病，也不信任不患病）。
